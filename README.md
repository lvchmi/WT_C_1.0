编译器使用简介
============
***
>**如何编译文件**

`comp.exe test`，test是自己要编译的文件名

>**支持语法**

* 声明

如：`int a;`，只支持单个的声明；支持类型`int`，`char`，`bool`.

* 赋值

支持，`+`，`-`，`*`，`/`，四种运算

* 条件语句

支持，`if{...}else{...}`，`for{...}`，`while{...}`，三种语句,可随意嵌套

* 比较语句

支持，`<`，`>`，`==`
